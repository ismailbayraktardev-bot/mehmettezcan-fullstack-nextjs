# Teknoloji Stack

## Core
- Next.js 15.0.3
- React 19.x
- TypeScript 5.x
- Node.js 20+

## Styling & Animation
- Tailwind CSS 3.4
- Framer Motion 11.x
- Lucide Icons

## CMS & Content
- Sanity Studio 3.x
- Next-Sanity 9.x
- Portable Text

## Forms & Email
- React Hook Form 7.x
- Zod (validation)
- Resend (email API)

## Development Tools
- ESLint
- Prettier
- Husky (git hooks)

## Deployment
- Vercel
- Sanity Cloud

## Package Install Command
```bash
npm install framer-motion lucide-react react-hook-form @hookform/resolvers zod sanity next-sanity @sanity/vision resend @react-email/components
```
